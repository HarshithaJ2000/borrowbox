06
Proposed Solution: QR Code Scan-In / Scan-Out System
Every item gets a unique QR code. Every checkout and return is scanned. Real-time inventory truth — always.
Step 01
QR Tag Every Item
Each item across all 12 hubs receives a unique QR code label. Scanned once to register into the database with category, hub, and baseline condition.
Step 02
Scan Out at Checkout
Staff scan the item QR code when a patron picks up. Patron ID is linked. Checkout time, expected return date, and item condition are recorded instantly.
Step 03
Scan In at Return
On return, staff scan the item again and log condition. System flags discrepancies against the checkout condition automatically and flags damage.
Step 04
Alert on Overdue
If an item is not scanned back in by the return deadline, the system notifies the patron and flags the item as potentially missing — enabling early recovery.
